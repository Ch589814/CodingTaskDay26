 **Project Description**

This project demonstrates the use of:

Object-Oriented Programming (OOP)

User input handling

Java Streams API

Grouping and counting data

The program allows the user to enter multiple people (name, age, and city), stores them in a List<Person>, and then uses Java Streams to group and analyze the data.

 **Class Structure:**
1️. Person Class

Contains:

name (String)

age (int)

city (String)

Includes:

Constructor

Getters and Setters

2️. Main Class

Responsible for:

Asking the user how many people to enter

Collecting name, age, and city

Storing data in List<Person>

Using Streams to:

Group people by age and display their names

**Features Implemented**

* Object-Oriented Design
* User Input using Scanner
* Java Streams
* Collectors.groupingBy()
* Collectors.mapping()
* Collectors.counting()

Group people by city and count how many people are in each city

Bonus: Changing the name of the second entered person
